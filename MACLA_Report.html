📋 Overview

MACLA is a domain-agnostic continual learning framework that enables LLM-based agents to learn from experience through structured procedural memory. Unlike traditional approaches that require extensive fine-tuning or per-step LLM reasoning, MACLA achieves state-of-the-art performance by:

Bayesian Selection Procedural Memory Meta-Procedures Contrastive Learning Hierarchical Memory Frozen LLM

The framework decouples reasoning from learning by maintaining a frozen LLM while performing all adaptation in an external hierarchical procedural memory. This enables sample-efficient, interpretable, and continually improving agents without LLM parameter updates.

⚙️ Key Algorithmic Components

🎲 Bayesian Selection

Uncertainty-aware procedure ranking using Beta posterior distributions:

# Beta-Bernoulli posterior success_rate = α / (α + β) variance = (α × β) / (n² × (n + 1)) # Expected utility scoring score = success_rate × reward_expected + exploration_bonus × √variance

Largest gain component — removing it drops accuracy by 9.1%

🔄 Contrastive Refinement

Quality improvement by contrasting success vs failure contexts:

# Extract discriminative patterns for procedure in memory: success_ctx = get_success_contexts(proc) failure_ctx = get_failure_contexts(proc) patterns = extract_discriminative( success_ctx, failure_ctx ) refine_procedure(proc, patterns)

Requires min 5 success + 5 failure examples per procedure

🏗️ Meta-Procedure Learning

Hierarchical composition of atomic procedures:

# Meta-procedure composition meta_proc = MetaProcedure( goal_meta="compound_task", sub_procedures=["cooling", "placement"], composition_policy={ "order": "sequential", "dependencies": { "placement": ["cooling"] } } )

Critical for unseen tasks — removing drops unseen by 11.9%

🧊 Frozen LLM Strategy

All adaptation happens externally — no parameter updates:

# Build memory in 56 seconds # 2,851 trajectories -> 187 procedures # Compression: 93.4% agent = LLMMACLAAgent( N_a=2000, # Atomic N_p=200, # Procedural N_m=50, # Meta-procedural llm_model="llama2:7b", use_llm=True )

2,800× faster than LLM parameter-training baselines

🎯 Key Findings
  • ✓ Bayesian selection provides the largest single gain — removing it drops accuracy by 9.1%
  • ✓ Meta-procedures are critical for unseen tasks — removing drops unseen by 11.9%
  • ✓ Positive generalization gap of +3.1% on ALFWorld unseen tasks — better on novel scenarios
  • ✓ Memory compression: 2,851 trajectories → 187 procedures (93.4% reduction)
  • ✓ 78% procedure reuse rate — most learned skills are actively used
  • ✓ 56 seconds to build full memory — 2,800× faster than fine-tuning baselines
  • ✓ All components contribute synergistically — no single ablation matches the full system
