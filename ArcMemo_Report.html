🧠 ArcMemo Dataset Report

Abstract Reasoning Composition with Lifelong LLM Memory • GitHub • arXiv:2509.04439
Generated 2026-05-18 16:23

ArcMemo introduces a concept-level memory framework for LLMs that moves beyond instance-based memory entries toward reusable, modular abstractions distilled from solution traces. Concepts are selectively retrieved and integrated into prompts, enabling test-time continual learning without weight updates. Evaluated on ARC-AGI, it yields a 7.5% relative gain over a strong no-memory baseline, with performance scaling with inference compute.
